CHECKBOX:
ROOT=HKLM
KEY=SOFTWARE\Microsoft\Windows\CurrentVersion\Policies\System
NAME=EnableLUA
TYPE=REG_DWORD
VALUE=0

CHECKED:
Reg.exe add "HKLM\SOFTWARE\Microsoft\Windows\CurrentVersion\Policies\System" /v "EnableLUA" /t REG_DWORD /d "0" /f

UNCHECKED:
Reg.exe add "HKLM\SOFTWARE\Microsoft\Windows\CurrentVersion\Policies\System" /v "EnableLUA" /t REG_DWORD /d "1" /f

DESCRIPTION:
Данный твик отключает контроль учетных записей (UAC), снижая ограничения безопасности, что может устранить ошибки с кодом 0x00002000, возникающие при запуске некоторых приложений.
При включении твика параметр EnableLUA устанавливается в 0.
При отключении – возвращается к стандартному значению (1).
Обратите внимание, что отключение UAC снижает уровень защиты системы.

EN_TITLE:
0x00002000 Disable User Account Control (UAC)

EN_DESCRIPTION:
This tweak disables User Account Control (UAC), reducing security restrictions, which may resolve errors like 0x00002000 that occur when launching certain apps. Enabling it sets EnableLUA to 0. Disabling it restores the default (1). Note that disabling UAC lowers the system's protection level.
