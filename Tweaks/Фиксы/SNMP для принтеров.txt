CHECKBOX:
ROOT=HKLM
KEY=SYSTEM\CurrentControlSet\Control\Print
NAME=DisableSNMP
TYPE=REG_DWORD
VALUE=1

CHECKED:
Reg.exe add "HKLM\SYSTEM\CurrentControlSet\Control\Print" /v "DisableSNMP" /t REG_DWORD /d "1" /f

UNCHECKED:
Reg.exe add "HKLM\SYSTEM\CurrentControlSet\Control\Print" /v "DisableSNMP" /t REG_DWORD /d "0" /f

DESCRIPTION:
Данный твик отключает SNMP-оповещения для подсистемы печати.
При включении твика значение параметра DisableSNMP устанавливается в 1, что предотвращает некорректное обновление статуса принтеров и может устранить ошибки с кодом 0x00000C22.
При отключении значение возвращается к стандартному (0), восстанавливая работу SNMP.
Используйте данный твик, если наблюдаются проблемы с отображением статуса принтеров.

EN_TITLE:
Disable SNMP for Printers

EN_DESCRIPTION:
This tweak disables SNMP notifications for the print subsystem. Enabling it sets DisableSNMP to 1, preventing incorrect printer status updates and potentially fixing error 0x00000C22. Disabling it restores the default behavior (0). Use this if you have issues with printer status display.

TR_TITLE:
