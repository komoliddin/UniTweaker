CHECKBOX:
ROOT=HKLM
KEY=SYSTEM\CurrentControlSet\Services\BITS
NAME=Start
TYPE=REG_DWORD
VALUE=4

CHECKED:
Reg.exe add "HKLM\SYSTEM\CurrentControlSet\Services\BITS" /v "Start" /t REG_DWORD /d "4" /f

UNCHECKED:
Reg.exe add "HKLM\SYSTEM\CurrentControlSet\Services\BITS" /v "Start" /t REG_DWORD /d "3" /f

DESCRIPTION:
Данный твик отключает службу BITS, которая обеспечивает фоновую загрузку обновлений Windows.
При включении твика параметр Start для BITS устанавливается в 4 (служба отключена), а при отключении – возвращается к значению 3 (вручную).
Это может помочь устранить проблемы, связанные с некорректной работой обновлений (например, ошибка 0x80244018).

EN_TITLE:
0x80244018 Disable Background Intelligent Transfer Service (BITS)

EN_DESCRIPTION:
This tweak disables the BITS service, which provides background downloading of Windows updates. Enabling it sets the Start parameter for BITS to 4 (disabled). Disabling it restores it to 3 (manual). This can help resolve issues related to improper Windows Update behavior (e.g., error 0x80244018).

DE_TITLE:
0x80244018 Hintergrundübertragungsdienst deaktivieren (BITS)

DE_DESCRIPTION:
Dieser Tweak deaktiviert den BITS-Dienst, der das Herunterladen von Windows-Updates im Hintergrund ermöglicht. Die Aktivierung setzt den Start-Parameter für BITS auf 4 (deaktiviert). Die Deaktivierung stellt ihn auf 3 (manuell) zurück. Dies kann helfen, Probleme im Zusammenhang mit fehlerhaftem Windows Update-Verhalten zu beheben (z. B. Fehler 0x80244018).

FR_TITLE:
0x80244018 Désactiver le service de transfert intelligent en arrière-plan (BITS)

FR_DESCRIPTION:
Ce tweak désactive le service BITS, qui assure le téléchargement en arrière-plan des mises à jour Windows. L'activation définit le paramètre Start pour BITS sur 4 (désactivé). La désactivation le restaure sur 3 (manuel). Cela peut aider à résoudre les problèmes liés au comportement incorrect de Windows Update (par exemple, l'erreur 0x80244018).

ES_TITLE:
0x80244018 Desactivar el Servicio de transferencia inteligente de fondo (BITS)

ES_DESCRIPTION:
Este ajuste desactiva el servicio BITS, que proporciona la descarga en segundo plano de actualizaciones de Windows. Al activarlo, el parámetro Start para BITS se establece en 4 (deshabilitado). Al desactivarlo se restaura a 3 (manual). Esto puede ayudar a resolver problemas relacionados con el comportamiento incorrecto de Windows Update (por ejemplo, el error 0x80244018).

TR_TITLE:
0x80244018 Arka Plan Akıllı Aktarım Hizmetini Devre Dışı Bırak (BITS)

TR_DESCRIPTION:
Bu ayar, Windows güncellemelerinin arka planda indirilmesini sağlayan BITS hizmetini devre dışı bırakır. Etkinleştirildiğinde BITS için Start parametresini 4 (devre dışı) olarak ayarlar. Devre dışı bırakılması 3 (manuel) değerine geri getirir. Bu, hatalı Windows Güncelleme davranışıyla ilgili sorunların (örneğin, 0x80244018 hatası) çözülmesine yardımcı olabilir.
