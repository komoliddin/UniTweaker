CHECKBOX:
ROOT=HKLM
KEY=SOFTWARE\Policies\Microsoft\Windows NT\Terminal Services
NAME=fDisableRemoteAssistance
TYPE=REG_DWORD
VALUE=1

CHECKED:
Reg.exe add "HKLM\SOFTWARE\Policies\Microsoft\Windows NT\Terminal Services" /v "fDisableRemoteAssistance" /t REG_DWORD /d "1" /f

UNCHECKED:
Reg.exe add "HKLM\SOFTWARE\Policies\Microsoft\Windows NT\Terminal Services" /v "fDisableRemoteAssistance" /t REG_DWORD /d "0" /f

DESCRIPTION:
Данный твик отключает функцию удалённой помощи (Remote Assistance) в Windows.
При включении твика параметр fDisableRemoteAssistance устанавливается в 1, что предотвращает возможность удалённого подключения для оказания помощи, что может устранить ошибки (например, 0x0000F000).
При отключении значение возвращается к 0, позволяя использовать удалённую помощь.

EN_TITLE:
0x0000F000 Disable Remote Assistance

EN_DESCRIPTION:
This tweak disables the Remote Assistance feature in Windows. Enabling it sets fDisableRemoteAssistance to 1, preventing remote connections for assistance, which can resolve errors (e.g., 0x0000F000). Disabling it restores the value to 0, allowing remote assistance.

UZ_TITLE:
0x0000F000 Masofaviy yordamni o'chirish

UZ_DESCRIPTION:
Ushbu tvik Windows-da masofaviy yordam (Remote Assistance) funksiyasini o'chiradi. Tvik yoqilganda fDisableRemoteAssistance parametri 1 ga o'rnatiladi, bu yordam ko'rsatish uchun masofadan ulanish imkoniyatini oldini oladi va ba'zi xatolarni (masalan, 0x0000F000) bartaraf etishi mumkin. O'chirilganda qiymat 0 ga qaytadi.

DE_TITLE:
0x0000F000 Remoteunterstützung deaktivieren

DE_DESCRIPTION:
Dieser Tweak deaktiviert die Remoteunterstützungsfunktion in Windows. Die Aktivierung setzt fDisableRemoteAssistance auf 1, wodurch Remoteverbindungen für Unterstützung verhindert werden, was Fehler (z. B. 0x0000F000) beheben kann. Die Deaktivierung stellt den Wert auf 0 zurück.

FR_TITLE:
0x0000F000 Désactiver l'assistance à distance

FR_DESCRIPTION:
Ce tweak désactive la fonction d'assistance à distance dans Windows. L'activation définit fDisableRemoteAssistance sur 1, empêchant les connexions à distance pour l'assistance, ce qui peut résoudre des erreurs (par exemple, 0x0000F000). La désactivation restaure la valeur à 0.

ES_TITLE:
0x0000F000 Desactivar Asistencia remota

ES_DESCRIPTION:
Este ajuste desactiva la función de asistencia remota en Windows. Al activarlo, fDisableRemoteAssistance se establece en 1, lo que impide las conexiones remotas для asistencia, lo que puede resolver errores (por ejemplo, 0x0000F000). Al desactivarlo se restaura el valor a 0.

TR_TITLE:
0x0000F000 Uzaktan Yardımı Devre Dışı Bırak

TR_DESCRIPTION:
Bu ayar Windows'ta Uzaktan Yardım özelliğini devre dışı bırakır. Etkinleştirildiğinde fDisableRemoteAssistance değerini 1 olarak ayarlar ve yardım için uzaktan bağlantıları engeller; bu durum bazı hataları (örneğin, 0x0000F000) çözebilir. Devre dışı bırakılması değeri 0'a geri getirir.
