CHECKBOX:
ROOT=HKLM
KEY=SYSTEM\CurrentControlSet\Services\BITS
NAME=Start
TYPE=REG_DWORD
VALUE=4

CHECKED:
Reg.exe add "HKLM\SYSTEM\CurrentControlSet\Services\BITS" /v "Start" /t REG_DWORD /d "4" /f

UNCHECKED:
Reg.exe add "HKLM\SYSTEM\CurrentControlSet\Services\BITS" /v "Start" /t REG_DWORD /d "3" /f

DESCRIPTION:
Данный твик отключает службу BITS, которая обеспечивает фоновую загрузку обновлений Windows.
При включении твика параметр Start для BITS устанавливается в 4 (служба отключена),
а при отключении – возвращается к значению 3 (вручную).
Это может помочь устранить проблемы, связанные с некорректной работой обновлений.

EN_TITLE:
Disable Background Intelligent Transfer Service (BITS)

EN_DESCRIPTION:
This tweak disables the BITS service, which provides background downloading of Windows updates. Enabling it sets the Start parameter for BITS to 4 (disabled). Disabling it restores it to 3 (manual). This can help resolve issues related to improper Windows Update behavior.

DE_TITLE:
Hintergrundübertragungsdienst deaktivieren (BITS)

DE_DESCRIPTION:
Dieser Tweak deaktiviert den BITS-Dienst, der das Herunterladen von Windows-Updates im Hintergrund ermöglicht. Die Aktivierung setzt den Start-Parameter für BITS auf 4 (deaktiviert). Die Deaktivierung stellt ihn auf 3 (manuell) zurück. Dies kann helfen, Probleme im Zusammenhang mit fehlerhaftem Windows Update-Verhalten zu beheben.

FR_TITLE:
Désactiver le service de transfert intelligent en arrière-plan (BITS)

FR_DESCRIPTION:
Ce tweak désactive le service BITS, qui assure le téléchargement en arrière-plan des mises à jour Windows. L'activation définit le paramètre Start pour BITS sur 4 (désactivé). La désactivation le restaure sur 3 (manuel). Cela peut aider à résoudre les problèmes liés au comportement incorrect de Windows Update.

ES_TITLE:

TR_TITLE:
Arka Plan Akıllı Aktarım Hizmetini Devre Dışı Bırak (BITS)

TR_DESCRIPTION:
Bu ayar, Windows güncellemelerinin arka planda indirilmesini sağlayan BITS hizmetini devre dışı bırakır. Etkinleştirildiğinde BITS için Start parametresini 4 (devre dışı) olarak ayarlar. Devre dışı bırakılması 3 (manuel) değerine geri getirir. Bu, hatalı Windows Güncelleme davranışıyla ilgili sorunların çözülmesine yardımcı olabilir.
