CHECKBOX:
ROOT=HKLM
KEY=SYSTEM\CurrentControlSet\Control\Session Manager\Memory Management\PrefetchParameters
NAME=EnablePrefetcher
TYPE=REG_DWORD
VALUE=0

CHECKED:
Reg.exe add "HKLM\SYSTEM\CurrentControlSet\Control\Session Manager\Memory Management\PrefetchParameters" /v "EnablePrefetcher" /t REG_DWORD /d "0" /f

UNCHECKED:
Reg.exe add "HKLM\SYSTEM\CurrentControlSet\Control\Session Manager\Memory Management\PrefetchParameters" /v "EnablePrefetcher" /t REG_DWORD /d "3" /f

DESCRIPTION:
Этот твик отключает Windows Prefetcher.
При включении твика значение параметра EnablePrefetcher устанавливается в 0, что отключает предварительное кэширование,
а при отключении – возвращается к значению 3, обеспечивая оптимальную работу Prefetcher.
Это может помочь устранить ошибки и проблемы с запуском приложений (например, 0x00007000).

EN_TITLE:
Disable Windows Prefetcher

EN_DESCRIPTION:
This tweak disables the Windows Prefetcher. Enabling it sets EnablePrefetcher to 0, disabling pre-caching. Disabling it restores the value to 3, ensuring optimal Prefetcher performance. This can help resolve errors and issues with app launches (e.g., error 0x00007000).
