CHECKBOX:
ROOT=HKLM
KEY=SOFTWARE\Microsoft\Windows\Windows Error Reporting
NAME=Disabled
TYPE=REG_DWORD
VALUE=1

CHECKED:
Reg.exe add "HKLM\SOFTWARE\Microsoft\Windows\Windows Error Reporting" /v "Disabled" /t REG_DWORD /d "1" /f

UNCHECKED:
Reg.exe add "HKLM\SOFTWARE\Microsoft\Windows\Windows Error Reporting" /v "Disabled" /t REG_DWORD /d "0" /f

DESCRIPTION:
Данный твик отключает службу Windows Error Reporting, что может помочь уменьшить количество всплывающих уведомлений об ошибках с кодами, начинающимися с 0x00004000.
При включении твика значение параметра Disabled устанавливается в 1, отключая отчёты.
При отключении – возвращается к стандартному значению (0).
Учтите, что отключение отчётов может затруднить диагностику системных проблем.

EN_TITLE:
Disable Windows Error Reporting

EN_DESCRIPTION:
This tweak disables the Windows Error Reporting service, which can help reduce error pop-ups with codes starting with 0x00004000. Enabling it sets the Disabled value to 1. Disabling it restores the default (0). Note that disabling reports may make it difficult to diagnose system issues.

DE_TITLE:

DE_DESCRIPTION:
Dieser Tweak deaktiviert den Windows-Fehlerberichterstattungsdienst, was dazu beitragen kann, Fehlermeldungen mit Codes zu reduzieren, die mit 0x00004000 beginnen. TR_TITLE:
Windows Hata Bildirimini Devre Dışı Bırak

TR_DESCRIPTION:
Bu ayar, 0x00004000 ile başlayan hata kodlarına sahip hata pencerelerini azaltmaya yardımcı olabilecek Windows Hata Bildirimi hizmetini devre dışı bırakır.
