CHECKBOX:
ROOT=HKLM
KEY=System\CurrentControlSet\Services\WSearch
NAME=Start
TYPE=REG_DWORD
VALUE=4

CHECKED:
Reg.exe add "HKLM\SYSTEM\CurrentControlSet\Services\WSearch" /v "Start" /t REG_DWORD /d "4" /f

UNCHECKED:
Reg.exe add "HKLM\SYSTEM\CurrentControlSet\Services\WSearch" /v "Start" /t REG_DWORD /d "2" /f

DESCRIPTION:
Данный твик отключает службу индексатора Windows Search (WSearch).
При включении твика значение параметра Start устанавливается в 4, что отключает автоматический запуск службы,
а при отключении – возвращается к значению 2 (автоматический запуск).
Это может помочь решить проблемы с производительностью и ошибками, связанными с поисковым индексированием (например, 0x00006000).

EN_TITLE:
Disable Windows Search Indexer

EN_DESCRIPTION:
This tweak disables the Windows Search Indexer (WSearch) service. Enabling it sets the Start value to 4, disabling automatic service startup. Disabling it restores the automatic startup (2). This can help resolve performance issues and errors related to search indexing (e.g., error 0x00006000).

DE_TITLE:
Windows-Suchindexer deaktivieren

DE_DESCRIPTION:
Dieser Tweak deaktiviert den Windows-Suchindexerdienst (WSearch). Die Aktivierung setzt den Startwert auf 4, wodurch der automatische Dienststart deaktiviert wird. Die Deaktivierung stellt den automatischen Start (2) wieder her. Dies kann helfen, Leistungsprobleme und Fehler im Zusammenhang mit der Suchindizierung zu beheben (z. B. Fehler 0x00006000).

TR_TITLE:

TR_DESCRIPTION:
Bu ayar Windows Arama Dizin Oluşturucu (WSearch) hizmetini devre dışı bırakır. Etkinleştirildiğinde Başlangıç değerini 4 olarak ayarlar ve otomatik hizmet başlatmayı devre dışı bırakır.
