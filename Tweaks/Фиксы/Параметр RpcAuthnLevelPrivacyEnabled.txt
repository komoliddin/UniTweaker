CHECKBOX:
ROOT=HKLM
KEY=SYSTEM\CurrentControlSet\Control\Print
NAME=RpcAuthnLevelPrivacyEnabled
TYPE=REG_DWORD
VALUE=0

CHECKED:
Reg.exe add "HKLM\SYSTEM\CurrentControlSet\Control\Print" /v "RpcAuthnLevelPrivacyEnabled" /t REG_DWORD /d "0" /f

UNCHECKED:
Reg.exe add "HKLM\SYSTEM\CurrentControlSet\Control\Print" /v "RpcAuthnLevelPrivacyEnabled" /t REG_DWORD /d "1" /f

DESCRIPTION:
Этот твик управляет параметром RpcAuthnLevelPrivacyEnabled, который определяет уровень аутентификации RPC для подсистемы печати. При включении (чекбокс установлен) значение устанавливается в 0, что может снизить требования к безопасности и повысить совместимость с некоторыми приложениями. При отключении (чекбокс снят) значение меняется на 1, что усиливает уровень безопасности системы. Используйте данный твик с осторожностью, учитывая баланс между повышенной безопасностью и возможными проблемами совместимости.

EN_TITLE:
Manage RPC Authentication Level (Privacy)

EN_DESCRIPTION:
This tweak manages the RpcAuthnLevelPrivacyEnabled parameter, which defines the RPC authentication level for the print subsystem. When enabled (checkbox set), the value is set to 0, which may lower security requirements and improve compatibility with some apps. When disabled (checkbox cleared), the value changes to 1, strengthening system security. Use with caution.
