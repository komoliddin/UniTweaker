CHECKBOX:
ROOT=HKLM
KEY=SOFTWARE\Policies\Microsoft\Windows\DataCollection
NAME=AllowTelemetry
TYPE=REG_DWORD
VALUE=0

CHECKED:
Reg.exe add "HKLM\SOFTWARE\Policies\Microsoft\Windows\DataCollection" /v "AllowTelemetry" /t REG_DWORD /d "0" /f

UNCHECKED:
Reg.exe add "HKLM\SOFTWARE\Policies\Microsoft\Windows\DataCollection" /v "AllowTelemetry" /t REG_DWORD /d "1" /f

DESCRIPTION:
Данный твик отключает сбор телеметрии в Windows.
При включении твика параметр AllowTelemetry устанавливается в 0, что полностью отключает сбор данных. Это может помочь устранить некоторые ошибки конфиденциальности (например, 0x00009000).
При отключении значение возвращается к 1 (или более высокому, в зависимости от политики), позволяющему сбор телеметрических данных.
Используйте с осторожностью, так как отключение телеметрии может затруднить диагностику проблем.

EN_TITLE:
0x00009000 Disable Windows Telemetry

EN_DESCRIPTION:
This tweak disables telemetry collection in Windows. Enabling it sets AllowTelemetry to 0, completely turning off data collection. This can help resolve some privacy errors (e.g., 0x00009000). Disabling it restores the value to 1 (or higher depending on policy), allowing data collection. Use with caution, as disabling telemetry can make problem diagnostics difficult.

Telemetriyani o'chirish muammolarni aniqlashni qiyinlashtirishi mumkinligi sababli ehtiyotkorlik bilan foydalaning.

DE_TITLE:
0x00009000 Windows-Telemetrie deaktivieren

DE_DESCRIPTION:
Dieser Tweak deaktiviert die Telemetrieerfassung in Windows. Die Aktivierung setzt AllowTelemetry auf 0, wodurch die Datenerfassung vollständig abgeschaltet wird. Dies kann helfen, einige Datenschutzfehler (z. B. 0x00009000) zu beheben. Die Deaktivierung stellt den Wert auf 1 (oder höher, je nach Richtlinie) zurück. Mit Vorsicht verwenden, da das Deaktivieren der Telemetrie die Fehlerdiagnose erschweren kann.

TR_TITLE:
0x00009000 Windows Telemetrisini Devre Dışı Bırak

TR_DESCRIPTION:
Bu ayar Windows'ta telemetri toplamayı devre dışı bırakır. Etkinleştirildiğinde AllowTelemetry değerini 0 olarak ayarlar ve veri toplamayı tamamen kapatır. Bu, bazı gizlilik hatalarının (örneğin, 0x00009000) çözülmesine yardımcı olabilir. Devre dışı bırakılması değeri 1'e (veya ilkeye bağlı olarak daha yüksek bir değere) geri getirir. Telemetriyi devre dışı bırakmak sorun teşhisini zorlaştırabileceğinden dikkatli kullanın.
