CHECKBOX:
ROOT=HKLM
KEY=SYSTEM\CurrentControlSet\Services\Tcpip\Parameters
NAME=TcpTimedWaitDelay
TYPE=REG_DWORD
VALUE=30

CHECKED:
Reg.exe add "HKLM\SYSTEM\CurrentControlSet\Services\Tcpip\Parameters" /v "TcpTimedWaitDelay" /t REG_DWORD /d "30" /f

UNCHECKED:
Reg.exe add "HKLM\SYSTEM\CurrentControlSet\Services\Tcpip\Parameters" /v "TcpTimedWaitDelay" /t REG_DWORD /d "240" /f

DESCRIPTION:
Данный твик изменяет время ожидания закрытия TCP-соединений (TcpTimedWaitDelay).
При включении твика значение устанавливается в 30 секунд, что может помочь устранить ошибки с кодом 0x00005000, возникающие при долгом ожидании закрытия соединения.
При отключении значение возвращается к стандартному 240 секунд.
Используйте данный твик для оптимизации сетевых подключений, если наблюдаются проблемы с задержками или сбоем соединений.

EN_TITLE:
TCP Connection Timeout (TcpTimedWaitDelay)

EN_DESCRIPTION:
This tweak changes the TCP connection wait time (TcpTimedWaitDelay). Enabling it sets the value to 30 seconds, which can help resolve error 0x00005000 occurring during long connection closure waits. Disabling it restores the standard 240 seconds. Use this to optimize network connections if you experience latency or connection failure issues.

DE_TITLE:
