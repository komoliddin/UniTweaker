CHECKBOX:
ROOT=HKLM
KEY=SOFTWARE\Policies\Microsoft\Windows\DataCollection
NAME=AllowTelemetry
TYPE=REG_DWORD
VALUE=0

CHECKED:
Reg.exe add "HKLM\SOFTWARE\Policies\Microsoft\Windows\DataCollection" /v "AllowTelemetry" /t REG_DWORD /d "0" /f

UNCHECKED:
Reg.exe add "HKLM\SOFTWARE\Policies\Microsoft\Windows\DataCollection" /v "AllowTelemetry" /t REG_DWORD /d "1" /f

DESCRIPTION:
Данный твик отключает сбор телеметрии в Windows.
При включении твика параметр AllowTelemetry устанавливается в 0, что полностью отключает сбор данных.
При отключении значение возвращается к 1 (или более высокому, в зависимости от политики), позволяющему сбор телеметрических данных.
Используйте с осторожностью, так как отключение телеметрии может затруднить диагностику проблем.

EN_TITLE:
Disable Windows Telemetry

EN_DESCRIPTION:
This tweak disables telemetry collection in Windows. Enabling it sets AllowTelemetry to 0, completely turning off data collection. Disabling it restores the value to 1 (or higher depending on policy), allowing data collection. Use with caution, as disabling telemetry can make problem diagnostics difficult.

DE_TITLE:

TR_TITLE:
Windows Telemetrisini Devre Dışı Bırak

TR_DESCRIPTION:
Bu ayar Windows'ta telemetri toplamayı devre dışı bırakır. Etkinleştirildiğinde AllowTelemetry değerini 0 olarak ayarlar ve veri toplamayı tamamen kapatır.
