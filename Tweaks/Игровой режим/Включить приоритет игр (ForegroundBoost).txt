CHECKBOX:
ROOT=HKLM
KEY=SOFTWARE\Microsoft\Windows NT\CurrentVersion\Multimedia\SystemProfile
NAME=NetworkThrottlingIndex
TYPE=REG_DWORD
VALUE=ffffffff

CHECKED:
Reg.exe add "HKLM\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Multimedia\SystemProfile" /v "NetworkThrottlingIndex" /t REG_DWORD /d "ffffffff" /f
Reg.exe add "HKLM\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Multimedia\SystemProfile\Tasks\Games" /v "Priority" /t REG_DWORD /d "6" /f

UNCHECKED:
Reg.exe add "HKLM\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Multimedia\SystemProfile" /v "NetworkThrottlingIndex" /t REG_DWORD /d "10" /f
Reg.exe add "HKLM\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Multimedia\SystemProfile\Tasks\Games" /v "Priority" /t REG_DWORD /d "2" /f

DESCRIPTION:
Увеличивает приоритет игр и отключает ограничение сетевой скорости (Network Throttling).

EN_TITLE:
Increase Game Priority (ForegroundBoost)

EN_DESCRIPTION:
Increases game priority in Windows to reduce input lag and disables network speed limits, useful for online gaming.

DE_DESCRIPTION:
