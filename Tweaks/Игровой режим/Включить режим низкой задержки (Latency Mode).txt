CHECKBOX:
ROOT=HKLM
KEY=SOFTWARE\Microsoft\Windows\Dwm
NAME=OverlayTestMode
TYPE=REG_DWORD
VALUE=5

CHECKED:
Reg.exe add "HKLM\SOFTWARE\Microsoft\Windows\Dwm" /v "OverlayTestMode" /t REG_DWORD /d "5" /f
Reg.exe add "HKLM\SOFTWARE\Microsoft\Windows\Dwm" /v "UseOLEDTaskMode" /t REG_DWORD /d "1" /f

UNCHECKED:
Reg.exe add "HKLM\SOFTWARE\Microsoft\Windows\Dwm" /v "OverlayTestMode" /t REG_DWORD /d "0" /f
Reg.exe add "HKLM\SOFTWARE\Microsoft\Windows\Dwm" /v "UseOLEDTaskMode" /t REG_DWORD /d "0" /f

DESCRIPTION:
Включает специальный режим в оконных играх, который снижает задержку вывода кадров.

EN_TITLE:
Low Latency Mode (DWM)

EN_DESCRIPTION:
Enables a special mode for windowed games that reduces frame output latency and makes controls more responsive.
