CHECKBOX:
ROOT=HKCU
KEY=Control Panel\Desktop
NAME=WaitToKillAppTimeout
TYPE=REG_DWORD
VALUE=0

CHECKED:
Reg.exe add "HKCU\Control Panel\Desktop" /v "MenuShowDelay" /t REG_DWORD /d "0" /f
Reg.exe add "HKCU\Control Panel\Desktop" /v "ForegroundLockTimeout" /t REG_DWORD /d "0" /f
Reg.exe add "HKCU\Control Panel\Desktop" /v "LowLevelHooksTimeout" /t REG_DWORD /d "0" /f

UNCHECKED:
Reg.exe add "HKCU\Control Panel\Desktop" /v "MenuShowDelay" /t REG_DWORD /d "400" /f
Reg.exe add "HKCU\Control Panel\Desktop" /v "ForegroundLockTimeout" /t REG_DWORD /d "200000" /f
Reg.exe add "HKCU\Control Panel\Desktop" /v "LowLevelHooksTimeout" /t REG_DWORD /d "5000" /f

DESCRIPTION:
Этот твик уменьшает задержку переключения между окнами и делает отклик управления в играх быстрее.
