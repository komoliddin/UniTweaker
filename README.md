# UniTweaker (Universal Tweaker)

**UniTweaker** — это мощный, легкий и универсальный инструмент для тонкой настройки, оптимизации и персонализации ОС Windows. Программа спроектирована как гибкая оболочка, позволяющая управлять системой через визуальный интерфейс, при этом сохраняя прозрачность работы через текстовые файлы конфигурации.

## 🚀 Основные возможности

### 🛠 Мощный движок твиков
- **Поддержка множества форматов:** Выполнение скриптов `.reg`, `.bat`, `.cmd`, `.vbs`, `.ps1`.
- **Простая кастомизация:** Все твики хранятся в папке `Tweaks` в виде обычных текстовых файлов.
- **Интеллектуальный парсинг:** Автоматическое создание UI на основе содержимого файлов (заголовки, описания на разных языках).

### 📊 Продвинутый Дашборд
- **Мониторинг ресурсов:** Реальное время нагрузки, частоты и температур CPU, GPU. Мониторинг RAM и дискового пространства.
- **Сетевой монитор:** Отображение текущей скорости загрузки и отдачи (Download/Upload) в реальном времени.
- **Статус безопасности:** Быстрая проверка состояния UAC, Защитника Windows, Брандмауэра и Центра обновлений.
- **Умные рекомендации:** Динамические советы по улучшению производительности на основе текущего состояния системы.

### 🧠 Умные функции (Smart Logic)
- **Tray Agent:** Возможность сворачивания программы в системный трей для фоновой работы с поддержкой уведомлений.
- **Auto RAM Optimizer:** Интеллектуальное освобождение оперативной памяти при достижении заданного порога нагрузки.
- **Junk Monitor:** Фоновое отслеживание объема временных файлов с автоматической очисткой.
- **Background Defrag:** Автоматическая оптимизация (дефрагментация) HDD-дисков во время простоя системы.
- **Broken Shortcut Scanner:** Обнаружение нерабочих ярлыков на рабочем столе с уведомлением.

### 🧹 Модуль Очистки
- **Комплексный анализ:** Поиск и удаление кэша, временных файлов (Temp, Prefetch), логов и мусора браузеров.
- **Фоновый анализатор:** Оценка потенциально освобождаемого места в реальном времени.

### 🔍 Системная информация (Deep Scan)
- **Hardware:** Детальные данные о материнской плате, процессоре (вкл. сокет и кодовое имя), видеокартах (вкл. шину данных).
- **Память:** Определение типа ОЗУ, режима канальности и поддержка XMP-профилей.
- **Диски:** Проверка здоровья дисков, времени наработки (часы), количества циклов включения и типа накопителя (SSD/HDD).
- **Мониторы:** Определение модели, разрешения и частоты обновления.

### 📦 Менеджеры компонентов
- **Metro Manager:** Управление UWP-приложениями (удаление, сброс, поиск иконок).
- **Service Manager:** Оптимизация служб через 5 пресетов: *Default (Manual), Safe, Optimal, Extreme, Restore*. Умный предпросмотр изменений и автоматическая остановка служб при отключении.
- **Autorun Manager:** Полный контроль автозагрузки (реестр, задачи планировщика, службы).
- **OEM Editor:** Персонализация информации о производителе и модели ПК.
- **Cursor Manager:** Удобное управление курсорами Windows. Поддержка предпросмотра схем, анимированных курсоров, автоматический импорт .inf файлов с расширенным распознаванием ролей (вкл. Person и Pin) и большая библиотека популярных наборов (Bibata, Oreo, macOS и др.).

## 🎨 Интерфейс и UX
- **Native UI:** Чистый дизайн на Windows Forms с полноценной поддержкой **Темной темы**.
- **Smart Scaling:** Полная поддержка High DPI (100% - 200%) с динамическим изменением размера окна.
- **Multilingual:** Нативная поддержка 7 языков (RU, EN, UZ, DE, FR, ES, TR).
- **Favorites:** Быстрый доступ к часто используемым твикам через систему «Избранное».

## 🛠 Технологический стек
- **Язык:** C# (.NET Framework 4.8)
- **Технологии:** WinForms, WMI, Win32 API, PowerShell Integration.
- **Архитектура:** Модульная структура без внешних тяжелых зависимостей.
